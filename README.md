# EduVerse_website

## Project Title

**EduVerse Academy - Responsive Educational Website**

## Project Overview

EduVerse Academy is a responsive educational website developed as part of the **MANTRA 2026 Summer School Frontend Website Assignment**. The project demonstrates frontend web development using **HTML5**, **CSS3**, and **JavaScript**.

The website is designed to provide information about an educational institution in a simple, clean, and user-friendly manner. It includes multiple pages, responsive design, basic SEO implementation, and JavaScript interactivity.

# Description of Each File

## index.html

* Home page of the website.
* Contains the hero section.
* Includes navigation menu.
* Displays introductory content.
* Includes "Explore Courses" button.

## about.html

* Provides information about EduVerse Academy.
* Explains the institution's mission and objectives.

## courses.html

* Displays the educational courses offered.
* Uses responsive course cards.

## gallery.html

* Shows campus photographs and educational activities.
* Uses image cards with hover effects.

## contact.html

* Displays contact information.
* Contains a contact form.
* Connected with JavaScript for form submission alert.

## style.css

* Contains all website styling.
* Responsive layout using CSS Media Queries.
* Hero section styling.
* Navigation styling.
* Footer styling.
* Card layouts.
* Hover animations.
* Mobile responsiveness.

## script.js

* Adds JavaScript functionality.
* Displays a success message after submitting the contact form.
* Handles button interactions.
